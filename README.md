# DevOps Docker Projects Collection

This repository contains a collection of beginner-to-intermediate DevOps projects built using Docker and containerized architectures. The goal is to practice real-world DevOps skills such as containerization, multi-service applications, and backend integration.

# Projects Included

##  Highscore App (Node.js + PostgreSQL)
A containerized backend API that stores and manages highscores using PostgreSQL. Built with Node.js, Express, Docker, and Docker Compose. Focuses on REST API design, database persistence, and multi-container orchestration.

##  Flask App
A simple Flask-based application used to understand Python backend containerization with Docker. It demonstrates how to build and run Python web apps inside isolated containers.

##  Flask + Redis Project
A small project demonstrating integration between Flask and Redis. It focuses on caching concepts, fast data storage, and multi-service Docker communication.

# Key Learning Areas

- Docker fundamentals
- Docker Compose multi-service setup
- Backend containerization (Node.js & Python)
- Database integration (PostgreSQL & Redis)
- Persistent storage and volumes
- Environment variables and service networking
